## How To Use

For the firs time it is necessary to install node_modules:

```
npm install
```

Then you are able to run server:

```
npm run dev
```

To build projects:

```
npm run build
```

To use prettier:

```
npm run prettier
```

To use eslint:

```
npm run eslint
```
